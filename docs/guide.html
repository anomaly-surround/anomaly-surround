1 Prerequisites

Before installing Anomaly Surround, make sure you have:

  • Windows 10 or 11
  • Python 3.10 or newer — Download from python.org. During install, check "Add Python to PATH".
  • Equalizer APO — Download from SourceForge. This is the free, open-source audio engine that handles all DSP processing.
Important: Without Equalizer APO, the app can run but cannot apply any audio processing. Make sure to install it first.

2 Configure Equalizer APO

After installing Equalizer APO:

  1. The Configurator will open automatically after installation.
  2. Find your headphones in the device list (e.g., HyperX Cloud Alpha S).
  3. Check the box next to your headphones to enable processing on that device.
  4. Click OK and restart your computer.
Tip: If you need to re-open the Configurator later, find it in Start Menu > Equalizer APO > Configurator.

3 Install Anomaly Surround

Download the app and install its dependencies:

# Navigate to the app folder
cd anomaly-surround

# Run the setup script (installs Python dependencies)
setup.bat

# Or install manually
pip install pycaw comtypes

That's it for installation. The app is ready to use.

4 Launch the App

The easiest way to start: double-click launch.vbs in the app folder. That's it — the app opens cleanly with no extra windows.

That's all most users need! Just run launch.vbs and you're good to go. The options below are for advanced users.

Other ways to launch (optional):

Background / Tray Mode

Runs silently in your system tray. Auto-switches profiles when you launch games.

python main.py --tray

CLI Mode

Command-line interface for quick adjustments.

python main.py --cli status

5 Using the GUI

The GUI has five tabs:

  • Main — Select your active profile, toggle surround/HRTF/EQ on or off, and control master volume.
  • Equalizer — 10-band parametric EQ (31Hz to 16kHz). Load a preset or customize. Adjust preamp gain to prevent clipping.
  • Surround — Configure virtual 7.1 channel levels: Stereo Width, Center, LFE, Rear, Side, and Room Size.
  • Games — View and manage game-to-profile mappings (40+ built-in). Add custom mappings.
  • Settings — Auto-start, minimize-to-tray, game scan interval, and detected devices.
Tip: Use Save As... on the Main tab to save your configuration as a new profile.

6 CLI Commands Reference

Format: python main.py --cli <command>

CommandDescription
statusShow current status, active profile, and detected devices
profilesList all available profiles
apply <name>Apply a profile (e.g., apply fps)
eqView current EQ band settings
eq -b 3 -g 4.5Set EQ band 3 to +4.5 dB
surround on/offEnable or disable virtual surround
volume 75Set master volume to 75%
games listShow all game-to-profile mappings
games detectScan for currently running games
games add game.exe fpsMap a game executable to a profile
autostart on/offToggle Windows startup

7 Setting Up Per-Game Profiles

The app auto-detects 40+ games. To add your own:

  1. Find the game's .exe name — Open Task Manager while the game is running, find the process name.
  2. GUI: Go to the Games tab, type the exe name, select a profile, click Add.
  3. CLI: Run python main.py --cli games add mygame.exe fps

The app auto-switches when you launch the game and reverts when you close it.

8 Running on Startup (Optional)

  • GUI: Settings tab → check "Start with Windows"
  • CLI: python main.py --cli autostart on

This launches the app in tray mode when Windows starts.

! Using With In-Game Spatial Audio

Some games have built-in HRTF (Valorant, CS2, Overwatch). Using both the game's HRTF and Anomaly Surround's HRTF causes double-processing which sounds worse.

Rule of thumb: If a game has its own spatial audio, turn OFF Surround and HRTF in Anomaly Surround. Just use the EQ.
Tip: Create two profiles:
• "FPS EQ Only" — for games with built-in spatial audio
• "FPS Full" — for games without spatial audio
Map each game to the right one — the app auto-switches.

? Troubleshooting

  • "Equalizer APO not installed" — Install Equalizer APO and enable it for your headphones in the Configurator. Restart your PC.
  • "Permission denied" — Run the app as Administrator.
  • No sound difference — Open Equalizer APO's Configuration Editor and verify config.txt was updated.
  • Game not detected — Check the exact .exe name in Task Manager and add it manually.
  • Audio crackling — Lower the preamp gain in the Equalizer tab.
  • Audio mutes when opening a game — Enable "Auto-apply EQ when game detected" in the Games tab, or click Apply to force refresh.
