HRTF Spatial Audio

Head-Related Transfer Function processing simulates realistic 3D positional audio. Hear footsteps behind you, gunshots to your left — pinpoint accuracy through stereo headphones.

10-Band Parametric EQ

Full parametric equalizer from 31Hz to 16kHz with adjustable Q factor and preamp gain. Fine-tune every frequency to match your headphones and preferences.

Auto Game Detection

Installation & Usage Guide

Complete step-by-step instructions to get Anomaly Surround running on your system.

1 Prerequisites

Before installing Anomaly Surround, make sure you have:

  • Windows 10 or 11
  • Python 3.10 or newer — Download from python.org. During install, check "Add Python to PATH".
  • Equalizer APO — Download from SourceForge. This is the free, open-source audio engine that handles all DSP processing.
Important: Without Equalizer APO, the app can run but cannot apply any audio processing. Make sure to install it first.

2 Configure Equalizer APO

After installing Equalizer APO:

  1. The Configurator will open automatically after installation.
  2. Find your headphones in the device list (e.g., HyperX Cloud Alpha S).
  3. Check the box next to your headphones to enable processing on that device.
  4. Click OK and restart your computer.
Tip: If you need to re-open the Configurator later, find it in Start Menu > Equalizer APO > Configurator.

3 Install Anomaly Surround

Download the app and install its dependencies:

# Navigate to the app folder
cd anomaly-surround

# Run the setup script (installs Python dependencies)
setup.bat

# Or install manually
pip install pycaw comtypes

That's it for installation. The app is ready to use.

4 Launch the App

Anomaly Surround has three modes. Choose whichever fits your workflow:

GUI Mode (Recommended)

Full graphical interface with EQ sliders, surround controls, and game management.

python main.py

Or just double-click launch.bat.

Background / Tray Mode

Runs silently in the system tray. Monitors games and auto-switches profiles.

python main.py --tray

CLI Mode

Command-line interface for scripting and quick adjustments.

python main.py --cli status

5 Using the GUI

The GUI has five tabs:

  • Main — Select your active profile, toggle surround/HRTF/EQ on or off, and control master volume. You'll also see the status of Equalizer APO and the game monitor here.
  • Equalizer — 10-band parametric EQ with vertical sliders (31Hz to 16kHz). Load a preset (FPS, RPG, Music) or drag the sliders to customize. Adjust preamp gain to prevent clipping. Click Apply EQ when done.
  • Surround — Configure virtual 7.1 channel levels individually: Stereo Width, Center, LFE (sub-bass), Rear, Side channels, and Room Size. Toggle Windows Sonic spatial audio on or off.
  • Games — View all game-to-profile mappings (40+ built-in). Add your own custom mappings by entering the game's .exe filename and selecting a profile. The app auto-detects running games and switches profiles.
  • Settings — Toggle auto-start with Windows, minimize-to-tray behavior, game scan interval, and view detected audio devices.
Tip: Use Save As... on the Main tab to save your custom configuration as a new profile you can switch to anytime.

7 Setting Up Per-Game Profiles

Anomaly Surround automatically detects 40+ games and switches to the best profile. Here's how to add your own:

  1. Find the game's .exe name — Open Task Manager while the game is running, find the process name (e.g., mygame.exe).
  2. In the GUI: Go to the Games tab, type the exe name, select a profile, and click Add.
  3. In the CLI: Run python main.py --cli games add mygame.exe fps

When the game launches, the app detects it and switches profiles automatically. When you close the game, it reverts to your default profile.

Note: The game monitor scans for running processes every 5 seconds (configurable in Settings). This uses minimal CPU resources.

8 Running on Startup (Optional)

To have Anomaly Surround start automatically with Windows and run in the background:

  • GUI: Go to Settings tab and check "Start with Windows".
  • CLI: Run python main.py --cli autostart on

This creates a startup entry that launches the app in tray/background mode. To disable it later, uncheck the option or run autostart off.

! Using With In-Game HRTF / Spatial Audio

Many games have their own built-in HRTF or spatial audio (e.g., Valorant, CS2, Overwatch). Running both the game's HRTF and Anomaly Surround's HRTF at the same time means double-processing, which can make positional audio sound worse, not better.

Important: If a game already has good spatial audio, disable Surround and HRTF in Anomaly Surround for that game. Only use the EQ.

Best practice:

  • Game has built-in HRTF (Valorant, CS2, Overwatch, Apex Legends) — Turn OFF Surround and HRTF in Anomaly Surround. Just use the EQ to boost footstep frequencies and tweak the sound to your preference.
  • Game has no spatial audio — Turn ON Surround and HRTF in Anomaly Surround. The app will handle directional audio processing for you.
Tip: Create two profiles to handle this easily:
• "FPS EQ Only" — Surround OFF, HRTF OFF, just EQ tuned for footsteps. Map this to games with built-in spatial audio.
• "FPS Full" — Surround ON, HRTF ON. Map this to games without spatial audio.
Then assign each game to the right profile in the Games tab — the app auto-switches for you.

? Troubleshooting

  • "Equalizer APO not installed" — Make sure Equalizer APO is installed and you've enabled it for your headphones in the Configurator. Restart your PC after setup.
  • "Permission denied" — Run the app as Administrator. Equalizer APO's config files are in Program Files, which requires elevated permissions to write.
  • No sound difference after applying — Open Equalizer APO's own "Configuration Editor" and verify that the config.txt file has been updated. Make sure no other audio software is overriding it.
  • Game not auto-detected — Check the exact .exe name in Task Manager and add it manually via the Games tab or CLI.
  • Audio crackling or distortion — Lower the preamp gain in the Equalizer tab. High EQ boosts combined with loud audio can cause clipping.
  • Audio mutes when opening or closing a game — Some games take exclusive control of the audio device, which interrupts Equalizer APO. Make sure "Auto-apply EQ when game detected" is checked in the Games tab. This lets the app automatically re-apply your audio settings after the game finishes initializing. If audio is still muted, click "Apply" on the Equalizer or Surround tab to force a config refresh.
